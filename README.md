# 업경 - Mirror of Karma

> 죄를 지은 자들이 스스로의 업보를 직면하도록,
> 이름도 얼굴도 없는 자가 지옥을 걷는 이야기.

## 소개
동양 공포 탑뷰 로그라이트 게임.
7명의 보스를 처치하고 최종 보스 원겁을 쓰러뜨려라.

## 장르
- 탑뷰 로그라이트
- 동양 공포

## 개발 환경
- Unity 6.3 LTS
- C#

## 캐릭터
- 역사(力士): 맨몸. 무겁고 강한 스타일
- 표사(鏢師): 검. 날렵하고 빠른 스타일
- 술사(術士): 주술. 원거리 스타일
